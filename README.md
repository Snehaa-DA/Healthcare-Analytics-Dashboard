# Healthcare Analytics Dashboard

## Overview

This project analyzes 55,500 healthcare records using Microsoft Excel.

The dashboard was built using:

- Pivot Tables
- Pivot Charts
- KPIs
- Interactive Slicers

## Dashboard Preview

![Dashboard Overview](dashboard%20overview%20of%20healthcare%20analytics.png)

## Key Insights

- Obesity patients recorded the highest average billing amount (~25,806).
- Elective admissions accounted for the largest share of hospital admissions.
- Cigna covered the highest number of patients.
- Average hospital stay remained consistent across medical conditions (~15.5 days).
